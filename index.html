An Online Visualization Framework for Multi-Dimensional GeoSpatial Data





GeoVis applies visual analytics to Multi-Dimensional GeoSpatial data
using Web World Wind for efficient GeoVisualization of massive climate data

About the Project

Project Features

Cloud Environment

Deployed within a cloud environment to improve the accessibility of visualization services.

Animation Visualization

Providing fundamental visualization functions with 2D and 3D animation.

3D Volumetric Rendering

Providing 3D volumetric rendering, utilizing the emergent CUDA technique to improve the efficiency of the visualization pipeline.
